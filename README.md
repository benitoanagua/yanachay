# Landing page - Hosting Bolivia

Landing page especializada en ofrecer servicios de hosting web para el mercado boliviano.

## Stack Tecnológico

- **Framework**: SvelteKit con TypeScript
- **Styling**: Tailwind CSS para estilos utilitarios
- **Build Tool**: Vite para desarrollo y construcción
- **Deployment**: Adaptador estático para hosting universal
- **Componentes**: Arquitectura modular y reutilizable

## Objetivo del Proyecto

Proporcionar una solución de hosting web accesible para emprendedores, negocios y creadores en Bolivia, evitando las limitaciones de pagos internacionales en dólares que superan los topes de transacción locales.

## Características Principales

- Diseño moderno y responsive optimizado para conversiones
- Componentes modulares para fácil mantenimiento
- Performance optimizada con SvelteKit
- Estilos consistentes con sistema de diseño unificado

## Desarrollo

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview
```

## Estructura

- `src/lib/components/` - Componentes reutilizables (Card, Button, PricingCard, etc.)
- `src/routes/` - Páginas y layout principal
- Configuraciones de TypeScript y Tailwind CSS preconfiguradas

## Propuesta de Valor

Hosting anual en bolivianos con soporte local, configuración inicial sin costo y facturación en moneda local, diseñado específicamente para las necesidades del mercado boliviano.
